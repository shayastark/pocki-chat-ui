[Permalink: Getting Started](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#getting-started)

### 1\. Clone the Project

[Permalink: 1. Clone the Project](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#1-clone-the-project)

```
mkdir -p privy-next-farcaster-mini-app && curl -L https://github.com/privy-io/privy-examples/archive/main.tar.gz | tar -xz --strip=3 -C privy-next-farcaster-mini-app examples-main/examples/privy-next-farcaster-mini-app && cd privy-next-farcaster-mini-app
```

### 2\. Install Dependencies

[Permalink: 2. Install Dependencies](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#2-install-dependencies)

```
pnpm install
```

### 3\. Configure Environment

[Permalink: 3. Configure Environment](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#3-configure-environment)

Copy the example environment file and configure your Privy app credentials:

```
cp .env.example .env.local
```

Update `.env.local` with your Privy app credentials:

```
# Public - Safe to expose in the browser
NEXT_PUBLIC_PRIVY_APP_ID=your_app_id_here
NEXT_PUBLIC_PRIVY_SIGNER_ID=your_signer_id_here # optional
```

**Important:** Variables prefixed with `NEXT_PUBLIC_` are exposed to the browser. Get your credentials from the [Privy Dashboard](https://dashboard.privy.io/).

### 4\. Start Development Server

[Permalink: 4. Start Development Server](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#4-start-development-server)

```
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000/) in your browser to see the application.

## Core Functionality

[Permalink: Core Functionality](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#core-functionality)

### 1\. Farcaster Mini App Integration

[Permalink: 1. Farcaster Mini App Integration](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#1-farcaster-mini-app-integration)

Automatic login using Farcaster's miniapp SDK with signature-based authentication.

[`src/app/page.tsx`](https://github.com/privy-io/examples/blob/main/examples/privy-next-farcaster-mini-app/src/app/page.tsx)

```
import { useLoginToMiniApp } from "@privy-io/react-auth/farcaster";
import miniappSdk from "@farcaster/miniapp-sdk";

const { initLoginToMiniApp, loginToMiniApp } = useLoginToMiniApp();

// Initialize a new login attempt to get a nonce for the Farcaster wallet to sign
const { nonce } = await initLoginToMiniApp();

// Request a signature from Farcaster
const result = await miniappSdk.actions.signIn({ nonce: nonce });

// Send the received signature from Farcaster to Privy for authentication
await loginToMiniApp({
  message: result.message,
  signature: result.signature,
});
```

### 2\. Create Multi-Chain Wallets

[Permalink: 2. Create Multi-Chain Wallets](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#2-create-multi-chain-wallets)

Programmatically create embedded wallets for multiple blockchains. Supports Ethereum, Solana, Bitcoin, and more.

[`src/components/sections/create-a-wallet.tsx`](https://github.com/privy-io/examples/blob/main/examples/privy-next-farcaster-mini-app/src/components/sections/create-a-wallet.tsx)

```
import { useCreateWallet, useSolanaWallets } from "@privy-io/react-auth";
import { useCreateWallet as useCreateWalletExtendedChains } from "@privy-io/react-auth/extended-chains";

const { createWallet: createWalletEvm } = useCreateWallet();
const { createWallet: createWalletSolana } = useSolanaWallets();
const { createWallet: createWalletExtendedChains } =
  useCreateWalletExtendedChains();

// Create Ethereum wallet
createWalletEvm({ createAdditional: true });

// Create Solana wallet
createWalletSolana({ createAdditional: true });

// Create Bitcoin/other chain wallets
createWalletExtendedChains({ chainType: "bitcoin-segwit" });
```

### 3\. Send Transactions

[Permalink: 3. Send Transactions](https://github.com/privy-io/examples/tree/main/examples/privy-next-farcaster-mini-app#3-send-transactions)

Send transactions on both Ethereum and Solana with comprehensive wallet action support.

[`src/components/sections/wallet-actions.tsx`](https://github.com/privy-io/examples/blob/main/examples/privy-next-farcaster-mini-app/src/components/sections/wallet-actions.tsx)

import { useSendTransaction } from "@privy-io/react-auth";
