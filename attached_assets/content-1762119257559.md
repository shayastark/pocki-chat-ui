## [​](https://docs.privy.io/wallets/connectors/ethereum/integrations/ethers\#ethers)  Ethers

Privy is fully compatible with ethers.js. To get an ethers provider for a user’s connected wallet, first [find your desired wallet](https://docs.privy.io/wallets/wallets/get-a-wallet/get-connected-wallet) from the **`wallets`** array and switch it to your desired network, using the wallet’s **`switchChain`** method:

### [​](https://docs.privy.io/wallets/connectors/ethereum/integrations/ethers\#ethers-v5)  Ethers v5

```
const privyProvider = await wallet.getEthereumProvider();
const provider = new ethers.providers.Web3Provider(privyProvider);
```
