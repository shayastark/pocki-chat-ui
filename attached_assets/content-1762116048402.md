To determine if Privy has fully processed all external and embedded wallet connections, use the **`ready`** boolean returned by the **`useWallets`** hooks.

To prompt a user to connect an external wallet (on EVM networks or Solana) to your app, use the `connectWallet` method from the `useConnectWallet` hook.

### [​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets\#parameters)  Parameters

[​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets#param-description)

description

string

A description for the wallet connection prompt, which will be displayed in
Privy’s UI.

[​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets#param-wallet-list)

walletList

WalletListEntry\[\]

A list of \[wallet option\] that you would like
Privy to display in the connection prompt.

[​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets#param-wallet-chain-type)

walletChainType

'solana-only' \| 'ethereum-only' \| 'ethereum-and-solana'

Filter the login wallet options to only show wallets that support the specified
chain type.

### [​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets\#callbacks)  Callbacks

You can optionally register an onSuccess or onError callback on the useConnectWallet hook.

```
const {connectWallet} = useConnectWallet({
    onSuccess: ({wallet}) => {

    },

    },

```

[​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets#param-on-success)

onSuccess

({wallet: Wallet}) => void

An optional callback function that is called when a user successfully connects their wallet.

[​](https://docs.privy.io/wallets/connectors/usage/connecting-external-wallets#param-on-error)

onError

({error: string}) => void

An optional callback function that is called when a user exits the connection flow or there is an error.
